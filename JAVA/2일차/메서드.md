# 메서드
- 메서드(메소드, Method)는 클래스 혹은 인터페이스 등의 구성요소가 가지는 멤버(Member) 중 하나이며, 호출(Call) 가능한 동작 단위인 대상이다.
- 메서드 작성 시, 메서드의 이름에 따라 최소한의 역할만 하도록 로직을 작성하는 것이 중요하다.
- 명명법: 카멜 케이스
- 메서드의 구조는 아래와 같다.
    >```java
    > [접근 제한자] [abstract|final|static]? [반환타입|void] [메서드 이름] ([매개 변수,...]?) {
    >   [구현부]
    > } 
    >```
